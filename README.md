# SafeWalk Landing Page

Landing page académica para SafeWalk desarrollada con HTML, CSS y JavaScript básico.

## Archivos principales

- `index.html`: estructura de la landing page.
- `css/styles.css`: estilos, responsive y animaciones simples.
- `js/main.js`: menú móvil, alerta superior, barras animadas, botón volver arriba y validación del formulario.
- `img/`: imágenes usadas por la página.

## Cómo ejecutar

Abre `index.html` en el navegador o abrir el link de publicado en GitHub Pages.
